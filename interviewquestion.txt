NUMPY INTERVIEW QUESTIONS AND ANSWERS
=====================================

BASIC QUESTIONS
===============

Q1. What is NumPy?

Answer:
NumPy (Numerical Python) is an open-source Python library used for numerical
computing. It provides support for multidimensional arrays, mathematical
operations, linear algebra, statistics, and random number generation.

------------------------------------------------------------

Q2. Why is NumPy faster than Python Lists?

Answer:
1. Arrays are stored in contiguous memory locations.
2. Operations are implemented using optimized C code.
3. Supports vectorized operations.
4. Uses less memory than Python lists.

------------------------------------------------------------

Q3. Difference between Python List and NumPy Array?

Answer:
Python List:
- Slower
- Uses more memory
- Can store multiple datatypes
- Limited mathematical operations

NumPy Array:
- Faster
- Memory efficient
- Homogeneous datatype
- Supports vectorized operations

------------------------------------------------------------

Q4. What is a Vector?

Answer:
A one-dimensional array is called a Vector.

------------------------------------------------------------

Q5. What is a Matrix?

Answer:
A two-dimensional array consisting of rows and columns.

------------------------------------------------------------

Q6. What is a Tensor?

Answer:
An array with more than two dimensions.

------------------------------------------------------------

Q7. What are shape, ndim, size, and dtype?

Answer:

shape:
Returns dimensions of an array.

ndim:
Returns number of dimensions.

size:
Returns total number of elements.

dtype:
Returns datatype of elements.

RESHAPING QUESTIONS
===================

Q17. What does reshape() do?

Answer:
Changes the dimensions of an array without changing the data.

------------------------------------------------------------

Q18. Difference between flatten() and ravel()?

Answer:

flatten():
Returns a copy of the array.

ravel():
Returns a view whenever possible.

------------------------------------------------------------

Q19. Which is more memory efficient: flatten() or ravel()?

Answer:
ravel() because it generally returns a view instead of creating a copy.

MATHEMATICAL OPERATIONS
=======================

Q26. What is Vectorization?

Answer:
Performing operations on complete arrays without using explicit loops.

Benefits:
- Faster execution
- Cleaner code
- Better readability

------------------------------------------------------------

Q27. Why is Vectorization important?

Answer:
It removes Python loops and significantly improves performance.

------------------------------------------------------------

Q28. What is Broadcasting?

Answer:
Broadcasting allows NumPy to perform operations on arrays having
different shapes without manually reshaping them.

------------------------------------------------------------

Q29. What are the advantages of Broadcasting?

Answer:
- Faster computations
- Less memory usage
- Simpler code

ADVANCED QUESTIONS
==================

Q46. What is a View?

Answer:
A view shares memory with the original array.
Changes affect the original array.

------------------------------------------------------------

Q47. What is a Copy?

Answer:
A copy creates independent memory.
Changes do not affect the original array.

------------------------------------------------------------

Q48. Difference between Copy and View?

Answer:

View:
Shares memory with original array.

Copy:
Creates separate memory.

------------------------------------------------------------

Q49. What is Contiguous Memory?

Answer:
Memory locations stored sequentially, allowing faster access
and processing.

------------------------------------------------------------

Q50. Why is NumPy important in Machine Learning?

Answer:
1. Fast numerical computations.
2. Efficient memory usage.
3. Matrix operations.
4. Linear algebra support.
5. Foundation of Pandas, TensorFlow, PyTorch, and Scikit-Learn.
