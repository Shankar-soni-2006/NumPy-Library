
================
1. WHAT IS NUMPY?
================

+----------------------+------------------------------------------------------+
| Topic                | Description                                          |
+----------------------+------------------------------------------------------+
| NumPy                | Numerical Python library for numerical computing     |
| Purpose              | Fast array processing and mathematical operations    |
| Supports             | Arrays, Matrices, Tensors, Linear Algebra, Statistics|
| Used In              | Data Science, ML, AI, Scientific Computing           |
+----------------------+------------------------------------------------------+


2. WHY NUMPY?
=============

+----------------------+------------------------------------------------------+
| Advantage            | Description                                          |
+----------------------+------------------------------------------------------+
| Speed                | Faster than Python Lists                             |
| Memory Efficient     | Uses less memory                                     |
| Mathematical Ops     | Supports vectorized calculations                     |
| Multi-Dimensional    | Supports 1D, 2D, and ND arrays                       |
| Broadcasting         | Operations on different shaped arrays                |
| Ecosystem Support    | Base library for Pandas, TensorFlow, etc.            |
+----------------------+------------------------------------------------------+

NUMPY QUICK REVISION
====================

Array Creation      : array(), zeros(), ones(), full(), arange(), linspace()
Array Info          : shape, ndim, size, dtype
Reshaping           : reshape(), flatten(), ravel()
Sorting             : sort()
Filtering           : where(), Boolean Masking
Combining           : concatenate(), vstack(), hstack()
Math Operations     : +, -, *, /, %, **
Aggregation         : sum(), min(), max(), mean(), cumsum()
Linear Algebra      : dot(), norm(), @, .T
Random Numbers      : random(), randint(), rand(), randn()
Utilities           : unique(), copy(), astype()
File Operations     : save(), load()

Remember:
Python List  -> Multiplication repeats elements
NumPy Array  -> Multiplication performs element-wise operation
